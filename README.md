  # AI Email Triage System

AI-powered customer support automation system using n8n, FastAPI, Groq AI, Supabase PostgreSQL, Gmail APIs, and Discord alerts.

---

# Features

- Gmail-triggered automation
- AI ticket classification
- Urgency detection
- Department routing
- PostgreSQL persistence
- Supabase integration
- Discord escalation alerts
- FastAPI backend APIs

---

# Tech Stack

- n8n
- FastAPI
- PostgreSQL
- Supabase
- Groq API
- Gmail API
- Python
- JavaScript

---

# Architecture

Gmail Trigger
↓
JavaScript Cleanup
↓
FastAPI Backend
↓
AI Analysis
↓
Supabase PostgreSQL
↓
Conditional Routing
↓
Discord Alerts
